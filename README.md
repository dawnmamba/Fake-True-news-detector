## Overview

**VeritasML** is a comprehensive machine learning system for automated **fake news detection and classification**. The project implements and compares five distinct ML/DL approaches â€” from classical algorithms to state-of-the-art transformers â€” to identify misinformation in news articles. Built with rigorous preprocessing pipelines and evaluation metrics, **VeritasML demonstrates the effectiveness of different modeling techniques on real-world disinformation datasets**.

## Project Scope

- Multi-model fake news classification system (5 models: Logistic Regression, SVM, Random Forest, MLP, DistilBERT)
- Advanced text preprocessing and TF-IDF vectorization pipeline
- Comprehensive model evaluation and comparison framework
- Transfer learning implementation with transformer-based models
- Production-ready prediction interface for real-time news verification
